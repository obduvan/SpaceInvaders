#### Требования:
* Python версии не ниже 3.6
* PyQt5 c модулями из "requirements.txt"

#### Команда для запуска игры:
* python start.py

#### Управление:
* Передвижение: стрелки left, right
* Стрельба: space

#### Команда для установки пакетов:  
pip install -r requirements.txt
